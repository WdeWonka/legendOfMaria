# 💜 The Legend of Maria: Dance of Destiny

A romantic pixel-art adventure game — made with love for a 2-year anniversary.

## 🎮 How to Play

### Mobile
- **D-Pad** → Move
- **A** → Attack / Confirm / Talk
- **B** → Run
- **START** → Pause Menu
- **SELECT** → Memories Collection

### Keyboard
- **Arrow Keys** → Move
- **Z** → Attack / Confirm
- **X** → Run
- **Enter** → Pause
- **Shift** → Memories

## 🗺️ World

| Area | Description |
|------|-------------|
| Sunflower Town | Starting town — talk to everyone |
| Whispering Forest | Dark forest — defeat the Shadow Tailor |
| Crystal Lake Village | Lake village — more NPCs & memories |
| Crystal Cave | Glowing cave — defeat the Stone Gorilla |
| Championship Arena | Final destination — unlocked with both items |

## 💎 Collect

- **10 Memory Crystals** → Unlock heartfelt memories in SELECT menu
- **Coins** → Scattered throughout the world
- **Hearts** → Restore health

## 🏆 Items to Recover

1. **Championship Costume** → Defeat the Shadow Tailor in Whispering Forest
2. **Golden Competition Pole** → Defeat the Stone Gorilla in Crystal Cave

## 🚀 Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# From the project folder
vercel
```

No build step needed — pure HTML/CSS/JS.
